What I do

I provide comprehensive testing services to ensure software is reliable, secure, and user-friendly. My quality assurance expertise spans Manual Testing, where I carefully examine applications to identify bugs and verify that all features work correctly; Automation Testing, using specialized tools and frameworks to run tests faster, more accurately, and consistently; Performance Testing, evaluating how applications behave under different load conditions to ensure optimal performance; Security Testing, identifying potential vulnerabilities and risks to keep applications protected; and Usability Testing, ensuring applications are intuitive and deliver an excellent user experience. This comprehensive approach ensures quality at every level of the software development lifecycle.

I specialize in web scraping and data extraction to help businesses gather valuable information from websites efficiently and accurately. My expertise includes building custom scrapers using tools like Python, Beautiful Soup, Selenium, and Scrapy to extract data from various sources; implementing automated data collection systems that run on schedules to keep information up-to-date; handling dynamic content and JavaScript-heavy websites to ensure complete data capture; performing data cleaning and transformation to deliver structured, ready-to-use datasets; and ensuring ethical scraping practices that respect robots.txt, rate limits, and website terms of service. Whether it's competitor analysis, market research, or data aggregation, I deliver reliable scraping solutions tailored to your specific needs.

I specialize in web development, creating clean and functional websites that help individuals and businesses establish their online presence. My expertise includes building personal websites that tell your story; portfolio websites that showcase your work professionally; and simple web applications with both frontend and backend functionality. I work with HTML, CSS, and JavaScript to create responsive designs that look great on any device; implementing interactive features to enhance user experience; and ensuring clean, maintainable code that's easy to update and scale. Whether you need a landing page, a personal blog, or a custom web solution, I deliver websites tailored to your specific needs.

I am expanding my expertise in cybersecurity and penetration testing to help identify vulnerabilities and strengthen system security. My growing skillset includes vulnerability assessment to identify weaknesses in applications and networks; understanding common security threats like SQL injection, XSS, and authentication flaws; learning penetration testing methodologies to evaluate system defenses; exploring security testing tools used in the industry; and applying secure coding practices to build more resilient software. Leveraging my QA background, I approach security testing with a quality-focused mindset to help protect applications from potential threats.
